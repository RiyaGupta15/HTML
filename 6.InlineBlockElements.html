Block elements are the elements which have space line before and after the text and occupies the whole width of line even if it is not used for example paragraph and headings are block elements because they start from the next line leaving so much of space.

Inline elements are the elements that start next to where it ends and occupies only that much of space which is neede for example image and link are next to each other and not occupying all the space so they are inline elements.
